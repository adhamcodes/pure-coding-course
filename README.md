## Why This Course Exists

You said it yourself: *"I can't code a single line without AI. I feel blank."* That's not a talent problem — it's a **training** problem. AI did your thinking, so your thinking muscle never grew. This course grows it. By Day 30 you'll trust your own brain to solve problems.

## Why Python (and not JavaScript)?

Smart choice on your part. You'll get plenty of **JavaScript, TypeScript, and React** in your No-Plan-B journey. So here we use **Python** — for three reasons:
1. **It's the cleanest language for pure thinking** — minimal punctuation, reads almost like English, so you focus on *logic*, not syntax noise.
2. **It gives you range** — now you'll know two language families, which makes you a more versatile developer.
3. **It's the language of AI/ML** — so this doubles as a head start for your future ML & Deep Learning journey.

You only need a browser to start; you'll install Python properly on Day 5.

## How It Works (Daily Loop)

1. Open the current module guide (start: `Module_0_Foundations/Module_0_Guide.md`).
2. Open today's log in that module's `Logs/` folder.
3. **LEARN** the concept → **DO** the exercises by hand/typed yourself → pass the **Transition Condition**.
4. Write your practice code in the module's `Exercises/` folder (`.py` files).
5. Struggle is expected. Sit in it. That struggle is the muscle growing.

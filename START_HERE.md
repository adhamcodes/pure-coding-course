# START HERE 👋 (Read This First)

This explains how to use the Pure Coding course in the simplest way. Read this, then read `RULES_NO_AI.md`, then begin.

---

## What is this?

A **30-day course to teach you to code without AI** — and to actually understand what's happening under the hood. One lesson a day. By the end, you can sit at a blank screen and write a working program from your own brain.

## The big promise

Right now a blank code editor feels scary, like a blank exam page. In 30 days it'll feel like a blank notebook — a place where *you* make things. We get there by doing reps, by hand, every day.

---

## The 2 files you use each day

1. **The Module Guide** = the teacher. Explains today's idea + the exercises.
   - Example: `Module_0_Foundations/Module_0_Guide.md`
2. **The Daily Log** = your worksheet with check-boxes.
   - Example: `Module_0_Foundations/Logs/Day_01.md`

And one playground: each module's **`Exercises/`** folder — where you write your practice code.

---

## Your daily lesson — 3 parts

1. 🟦 **LEARN** — read today's concept in the guide. Take notes *in your own words* (not copy-paste).
2. 🟩 **DO** — solve the exercises yourself. Type every line. No AI. Struggle is allowed and expected.
3. 🟥 **TRANSITION CONDITION** — a small task you must do from memory to prove you got it. Can't do it? Repeat the day.

---

## How do I actually run code? (no setup needed to start)

For the first lessons you need **nothing but your browser**:
1. Open your web browser (Chrome, Edge, Firefox).
2. Press **F12** (or right-click → Inspect) to open Developer Tools.
3. Click the **Console** tab.
4. Type: `console.log("Hello, I am coding")` and press Enter.

That's it — you just ran code. Module 0 teaches you to set up a proper environment (Node + an editor) by Day 5.
